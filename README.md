
# **Nightclub City**

**בנה את אימפריית חיי הלילה שלך - מתחילים מקטן בפינה באלנבי, ועד לניהול ההפקות הגדולות במדינה**

---

## מהות המשחק

Nightclub city הוא משחק ניהול-אסטרטגיה בו המשאב העיקרי הוא כמובן כסף.
אתה ניצב כמנהל מועדון לילה קטן ולאט לאט מטפס בדרכך לנהל את חיי הלילה במדינת ישראל.
מניהול המוזיקה, אנשי צוות, והנדל"ן ועד לשיווק, מפורסמים וכל מה שיש לחיי הלילה להציע.
